## [1.0.0] - 2025-11-16

### Added
- **Live launch tracking** - Automatically detect launches within Â±2 hours
- **Embedded livestreams** - Watch launches directly in the app
- **Push notifications** - Get notified before launches (PWA)
- **Countdown timers** - Real-time countdowns
- **Calendar integration** - Add launches to calendar with one click
- **Filter by agency** - Filter launches by SpaceX, NASA, ESA, etc.
- **3-month view** - See all upcoming launches for the next 3 months
- **Rocket facts banner** - Rotating facts with NASA APOD
- **Offline support** - PWA with offline capabilities
- **Past launches page** - View historical launch data

- TypeScript 5
- Tailwind CSS 4
- SpaceX API v4 integration
- Launch Library 2 API integration

- Multi-layer caching (client + server)
- GitHub Actions CI/CD
